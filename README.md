# 贪吃蛇游戏

一个使用HTML5 Canvas开发的贪吃蛇网页游戏，具有现代化的设计和流畅的游戏体验。

## 游戏特点

- 🎯 **自动移动**：游戏开始时蛇会自动向右移动
- 👀 **生动设计**：蛇头带有白色眼睛，视觉效果更生动
- ⌨️ **双重控制**：支持键盘方向键 + 屏幕按钮
- 📊 **分数系统**：实时分数 + 本地最高分记录
- 🎨 **现代化设计**：深色主题，流畅动画效果
- 💾 **数据持久化**：自动保存最高分记录

## 如何开始

### 方法1：本地运行
1. 下载 `index.html` 文件
2. 直接在浏览器中打开

### 方法2：本地HTTP服务器
```bash
python -m http.server 8000
```
然后在浏览器中访问：http://localhost:8000

## 游戏控制

- **键盘方向键**：控制蛇的移动方向
- **屏幕按钮**：点击上下左右箭头控制
- **空格键**：重新开始游戏

## 游戏规则

1. 蛇会自动向前移动
2. 吃到红色食物获得分数
3. 蛇的身体会随着吃到的食物变长
4. 撞到边界或自己的身体游戏结束
5. 分数越高，游戏速度越快

## 技术实现

- **HTML5 Canvas**：游戏画面绘制
- **JavaScript**：游戏逻辑实现
- **CSS3**：现代化的样式设计
- **LocalStorage**：本地数据存储
